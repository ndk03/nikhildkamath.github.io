# Nikhil Kamath - Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript. Features a sleek dark/light theme, smooth animations, and an elegant user interface.

## Features

- **Dark/Light Theme Toggle** - Seamless theme switching with persistent preference
- **Smooth Animations** - Powered by AOS (Animate On Scroll) library
- **Typing Effect** - Dynamic typing animation in the hero section
- **Responsive Design** - Works beautifully on all devices
- **Interactive UI** - Cursor glow effect, animated skill bars, and more
- **Glassmorphism Design** - Modern frosted glass aesthetic
- **Scroll Progress Bar** - Visual indicator of page scroll position

## Sections

1. **Hero** - Eye-catching landing with animated gradient orbs
2. **About** - Personal introduction with stats and info cards
3. **Skills** - Tabbed interface showing technical expertise
4. **Experience** - Timeline of work history and education
5. **Contact** - Contact form and social links

## Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid, Animations)
- Vanilla JavaScript
- [AOS Library](https://michalsnik.github.io/aos/) - Scroll animations
- [Font Awesome](https://fontawesome.com/) - Icons
- [Google Fonts](https://fonts.google.com/) - Inter & Fira Code

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. Customize the content in `index.html`
4. Modify styles in `css/style.css`
5. Update functionality in `js/main.js`

## Customization

### Personal Information
Edit the HTML file to update:
- Name and tagline
- About section content
- Skills and proficiency levels
- Work experience and education
- Contact information and social links

### Colors
Modify CSS variables in `css/style.css`:
```css
:root {
    --accent-primary: #6366f1;
    --accent-secondary: #8b5cf6;
    --accent-tertiary: #ec4899;
}
```

### Profile Image
Replace the placeholder in the About section with your image.

## Deployment

This site is designed for GitHub Pages:
1. Push to your `username.github.io` repository
2. Enable GitHub Pages in repository settings
3. Your site will be live at `https://username.github.io`
